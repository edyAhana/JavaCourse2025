Объяснение

В Java блок try может иметь только один блок finally.
Допускается один try + один или несколько catch и не более одного finally.
Если написать два блока finally подряд:

try {
    ...
} finally {
    ...
} finally {  <- ошибка компиляции
    ...
}

Вывод

Два блока finally подряд невозможны.
Для нескольких завершающих действий используйте один finally с любым количеством инструкций.
